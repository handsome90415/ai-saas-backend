# AI Content Generator

AI-powered content creation tool for social media and e-commerce.

## Features

- Text generation for multiple platforms
- Image generation using DALL-E 3
- Traditional Chinese UI
- Multiple content styles

## Tech Stack

- Frontend: Next.js 14 + TypeScript + Tailwind CSS
- Backend: FastAPI + Python
- AI: OpenAI GPT-4 + DALL-E 3

## Quick Start

```bash
# Backend
cd backend
pip install -r requirements.txt
cp .env.example .env
python main.py

# Frontend
cd frontend
npm install
npm run dev
```
